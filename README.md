
# Oliver Ueno Portfolio

**Live site:** [oliverueno.site](https://oliverueno.site/)

## About

I enjoy turning ideas into useful products and working on ambitious projects with other students. My interests include full-stack development, entrepreneurship, student-led events, math, piano, soccer, and chess.

This portfolio is designed to quickly show what I have built, what I contributed, and what I am currently learning.

## Selected Work

### JecHacks

A free 12-hour online hackathon for high school students.

I co-founded the event and helped with sponsorship outreach, prizes, social media, and event planning. We secured more than $5,300 in funding and brought in over 120 student entrants.

[Visit JecHacks](https://www.jechacks.com/)

### SafeWalk

A pedestrian navigation project created during EurekaHacks. It compares routes using lighting and environmental safety information instead of only travel time.

I worked on routing, safety scoring, the responsive map and sidebar, deployment, and rapid bug fixing.

- [Live project](https://safewalk-eurekahacks.vercel.app/)
- [GitHub repository](https://github.com/olinho11/EurekaHacks26-SafeWalk)

### Three-Body Problem Simulator

An interactive physics simulator for exploring gravitational systems.

It includes RK4, Verlet, and Euler integration methods, presets, live analytics, chaos detection, trajectory prediction, and interactive body controls.

[Launch the simulator](https://3-body-problem.vercel.app/)

### ProjectPacket

A SaaS MVP that helps freelancers and teams collect files, information, and approvals from clients through private packet links.

It includes dashboards, templates, approval controls, passcodes, and expiry settings.

[Visit ProjectPacket](https://www.projectpacket.site/)

## Built With

The portfolio itself uses plain HTML, CSS, and JavaScript. It does not require a frontend framework or build step.

The site is deployed through Vercel.

## Running Locally

Clone the repository:

```bash
git clone https://github.com/olinho11/Personal-Portfolio.git
cd Personal-Portfolio
```

Start a local server:

```bash
python3 -m http.server 5176
```

Then open:

```text
http://localhost:5176/
```

## Verification

Check the JavaScript syntax:

```bash
node --check script.js
```

The repository also includes `verify_portfolio.py`, which checks important anchors, external-link security, image metadata, local assets, and accessibility basics.

It requires Pillow:

```bash
python3 -m pip install pillow
python3 verify_portfolio.py
```
